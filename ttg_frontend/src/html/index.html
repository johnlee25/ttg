The TimeTable Generator

By Dillon Dixon

This piece of software is designed to allow one to create a course time-table based on publicly available scheduling information provided for a university.

To begin, select your university from the list below. From there, select a course from the drop down list below, starting at department and then course name and then sections. More courses can be added after.

This software was designed originally for McMaster University, so the selection style is most similar to the one used by McMaster's registration system known as SOLAR. The timetable will be dynamically updated as you add more courses, and conflicts will show up if they are present.

If your university is not listed, but you know of a link for publicly accessible scheduling information, do not hesitate to contact us via one of the social networks below.

For the latest updates, like The Quintessential Timetable Generator on facebook.
